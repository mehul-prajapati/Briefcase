Hit the Escape key; that goes into command mode. Then you can type

  • :q to quit
  • :q! to quit without saving
  • :wq to write and quit
  • :qa to quit all

When you press :, a : will appear at the bottom of the screen.

Or you can press Escape ZZ (ShiftZZ) to write/save if the file was changed, then quit.

Or if you don't want to save changes you can use ZQ instead.

Vim has extensive help, so type Escape:helpReturn and you will have all your answers and even a neat tutorial.
